## Live Demo

[View Site](https://artgolwebdev.github.io/LP04/)

## Features

- Smooth scroll with section snapping
- Responsive design for all devices
- Interactive navigation
- Keyboard navigation support

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS + Framer Motion
- Vite build tool
